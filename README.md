# FBI-Wanted App

This project displays the FBI Most Wanted list using the official FBI API.

## Usage Notes
- If the FBI API is down or unreachable, the app will automatically use the local `mock-wanted.json` file for data.
- Sorting and search features are available for the wanted list.

## Deployment
- The site is deployed using GitHub Pages from the `master` branch and root directory.

## Features
- Search by name or alias
- Sort by name, reward, or date
- Fallback to mock data if API fails

## Project Files
- `index.html` — Main homepage
- `indexFBI.js` — App logic
- `indexFBI.css` — Styles
- `mock-wanted.json` — Mock data
